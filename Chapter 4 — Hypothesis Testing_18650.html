2.1 Point Estimation — The Setup

In Chapter 1 we established the basic problem: we observe a sample $X_1, \ldots, X_n \overset{\text{i.i.d.}}{\sim} F_\theta$ where the parameter $\theta$ is unknown. Now we face the question: how do we produce a specific numerical guess for $\theta$?

2.1.1 What Is a Point Estimator?

Definition · Point Estimator

A point estimator $\hat{\theta}_n$ of a parameter $\theta$ is any function $\hat{\theta}_n = T(X_1, \ldots, X_n)$ of the sample. It is a random variable. We measure how good an estimator is via its mean squared error:

$$\text{MSE}(\hat{\theta}_n) = \underbrace{\bigl(\mathbb{E}[\hat{\theta}_n] - \theta\bigr)^2}_{\text{bias}^2} + \underbrace{\text{Var}(\hat{\theta}_n)}_{\text{variance}}$$

2.1.2 Method of Moments

The Method of Moments (MoM) is the oldest estimation technique. It is based on a simple idea: set sample moments equal to population moments and solve for $\theta$.

Example · MoM for a Normal Distribution

Let $X_i \sim \mathcal{N}(\mu, \sigma^2)$. Setting sample moments equal to population moments gives: $$\hat{\mu} = \bar{X}, \qquad \hat{\sigma}^2 = \frac{1}{n}\sum (X_i - \bar{X})^2$$

💡
Key insight: When you minimize cross-entropy loss in machine learning, you are maximizing the log-likelihood. MLE is the foundational justification for most modern loss functions.

2.2 The Likelihood Function

2.2.1 Definition and Construction

Definition · Likelihood Function

The likelihood function $L(\theta;\mathbf{x})$ is defined as the joint density of the data, viewed as a function of the parameter $\theta$ with the data held fixed:

$$L(\theta;\mathbf{x}) = \prod_{i=1}^n f(x_i;\theta)$$

The MLE is the value that maximizes this: $\hat{\theta}_{\text{MLE}} = \operatorname*{arg\,max}_{\theta \in \Theta} L(\theta;\mathbf{x})$

Intuition · Likelihood vs. Probability

Likelihood is not the probability that $\theta$ equals some value. It tells us how "compatible" each candidate value is with what we observed.

2.2.2 The Log-Likelihood and Score Equation

Key Formula
$$\ell(\theta;\mathbf{x}) = \log L(\theta;\mathbf{x}) = \sum_{i=1}^n \log f(x_i;\theta)$$

Maximizing the log-likelihood is equivalent to maximizing the likelihood because log is a strictly increasing function.
